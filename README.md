# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/72dbf383-bb91-4e8e-bd3b-5ea2763759e7

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `XAI_API_KEY` in [.env.local](.env.local) to your xAI / Grok API key
3. Run the app:
   `npm run dev`
